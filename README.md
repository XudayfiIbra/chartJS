
# React With CharJS
## Clone the project
```js
git clone https://github.com/XudayfiIbra/chartJS.git
cd chartJS
```
### install dependencies
```js
npm install
```
## Installation for sperate project
### NPM
```js
npm install chart.js

```
### CDN
```js
https://cdnjs.com/libraries/Chart.js
```
### Start the server

```javascript
npm run dev
```
