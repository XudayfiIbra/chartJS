
# React With CharJS
## Install CharJS with npm
```js
git clone https://github.com/XudayfiIbra/chartJS.git
cd chartJS
```
### install dependencies
```js
npm install
```
## Installation only for Chartjs
### NPM
```js
npm install chart.js

```
### CDN
```js
https://cdnjs.com/libraries/Chart.js
```
### Available scripts

```javascript
npm start
npm test
npm run build
npm run inject
```
